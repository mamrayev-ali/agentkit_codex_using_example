# Frontend (T3 scaffold)

Angular shell scaffold for Decider.

## Commands

```bash
pnpm install
pnpm start
pnpm lint
pnpm test -- --run
pnpm build
```

## Route skeleton

- `/dashboard`
- `/dossiers`
- `/watchlist`
- `/**` (not found fallback)

## Environment files

- `src/environments/environment.ts`
- `src/environments/environment.production.ts`
